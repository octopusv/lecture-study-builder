# 大学授業 自動まとめ

大学の授業動画から、文字起こし、スライドOCR、詳細な授業ノート、
4択問題、暗記カード、教科別の学習Webサイトを生成するClaude Code向けワークスペースです。

特定の大学や教科には依存せず、任意の教科を追加して使えます。
元動画、文字起こし、OCR全文などの私的データはGitの管理対象外として扱います。

## 主な機能

- Whisperによる日本語音声認識
- Apple VisionによるスライドOCR
- 講義回別ノートと全講義の統合まとめ
- 講義内容に基づく4択問題と暗記カード
- PC・スマートフォン対応の教科別学習サイト
- 中断後も未完了工程から再開できる教科分析Skill

## クイックスタート

Apple Silicon搭載MacとClaude Codeを標準環境としています。

```text
/setup
/setup-course 01 教科名
```

作成された `教科別/01_教科名/videos/` に授業動画を配置し、解析を開始します。

```text
/analyze-course 01
```

教科ID、教科名、講義回数、各回の動画本数は
ローカルの `config/subjects.json` で管理します。

## データの扱い

`教科別/`、`config/subjects.json`、動画、文字起こし、OCR全文、生成途中の成果物は
リポジトリへ含めません。公開サイトを作る場合も、公開可能な問題や暗記カードだけを
許可リスト方式で別ディレクトリへ生成してください。

詳しい処理方針、品質基準、公開版と非公開版の分離方法は
`CLAUDE.md` を参照してください。
